健身方法论

发表于 2017-07-01 | | 本文总阅读量 次

起因

放出这篇文章时候，我已经完成健身了，体重从77 kg减到了 kg。所以亲身实践，值得尝试。

我在健身房呆过一年半，不是那种两天打鱼三天晒网的那种，除了过年回去，基本风雨无阻。也请过私人教练，自己也短暂的做过一段时间的教练，但后面因为发现因为自己也在上班，那么由于花在健身的时间太多了，导致根本没有时间做其它事，这样子只是练筋骨，没有练脑袋，这样子不行，而且我开始的锻炼目的已经达到，使得自己的精力充沛，当然还有个“借口”—搬家了，住的楼层太高了离健身房太远了……好吧，其实就是自己变懒了，然后就没然后了，体重就开始一骑绝尘去了。

由于我从来都不是为了减肥的目的去的，一开始的目的是为了增肌去的，效果非常好，一个月长10斤。但副作用就是再也没有降下来过，人到中年肚子圆……一开始我并不是很在乎，因为现在的我精力非常好，基本中午连午休都很少。

健身=长寿

当我知道这么一个事实，注意这是事实：

  1. 由于我们的饮食结构是以碳水化合物为主的，大部分国人，若不锻炼，保持这样的饮食到30岁之后，就会慢慢发胖（基础代谢降低）。
  2. 发胖不仅仅是皮下脂肪，还有内脏脂肪，而内脏脂肪超标，是很多脏器疾病的诱因。（BBC出过一个纪录片叫做解剖肥胖，太过重口味，建议一定要去看网址：http://www.iqiyi.com/v_19rr9if1ec.html

这就很严重了，这就意味着，不锻炼的话，不可避免就会短命……我怕死，真的怕，

而经常锻炼可以达到什么效果，我说的是正确的健身，有个住在新加坡的日本人50岁了，网络上流传着他的人鱼线，那身材体魄神态基本和三十多岁没有两样，胡子一刮，活脱一个小鲜肉……

非常不淡定了，

我必须重新捡起来健身者玩意了，在我的认知，健身应该是普通人最容易达成的成就，因为科学的饮食、科学的锻炼方法，都已经被验证了无数遍，包括国内外。虽然停止身体的锻炼，但学习健身的知识从来没有断过，现在基本有一整套锻炼的方法论，通过这个方法，我实现了每天只锻炼30 min，基本不做跑步等有氧运动，体重没有太大变化但脂肪都已经褪去，肌肉生成了。注意我每天是“只”练30 min。

这是什么概念，意思基本都不影响你其它的时间安排。这太重要了，好多人其实是没有时间导致没去锻炼，他们知道锻炼的好处。每天抽出30 min 投资自己的身体完全可以做到，即使那些加班狂魔。

短时间的锻炼时长可以让人更好的坚持，这是必然的。而这也是科学的，如果你有趣健身房呆过，只要你注意观察，就很容易发现，一般那些呆在那里一会跑下步，一会举下杠铃，一会吹水的人基本是那些“努力”健身的人，基本上他们每天都去，每天呆在那里都是超过2个小时，而那些一身腱子肉的人一般做几组该做的动作，做完就走，不拖泥带水。相比下健身的效率相差太大了，这太像我们读书的时候，学霸学习总是看过去举重若轻，而拖后腿的看过去是真的“努力”。

科学的锻炼确实有方法，现在的我只建议就做无氧运动就够了，因为这样子更合算，关于人体的肌肉有这个优势一定要知道：

你的肌肉含量越多，即便你一整天啥事都不干，你的肌肉功耗都比别人大。这被称之为静息代谢,

有氧运动也很好，但那很“小儿科”，或者说没啥技巧，每天跑5公里，只要坚持就可以了，这个我在大学时候就验证过了，确实有用，可是那时候代谢快，人到中年体力和精力的下降，如果没有习惯养成，基本这是一条死路，我真的不相信人的自制力……但幸亏，有更好的出路非常简单，根据以上的事实，我们需要做的是让身体开源，增肌，而这只需做无氧运动，而无氧健身其实只要做到这两方面，一是练到位，二是吃的对，做好这两方面，还会顺带解决无聊和晚上睡觉失眠等问题。

练到位

要努力做到这三点：

  1. 全身心投入
  2. 做到“力竭”
  3. 达到最大心律60%以上，要知道低于60% HRmax 的有氧运动，基本跟没做差不多，这也解释了上面的例子的原因。

健康人群的最大心率计算公式为：

HRmax（最大心率）=208-0.7×年龄

解释一下

练到位的主观感觉，是那种“再也不可能做一下动作”状态，非常缺氧。人的主观感觉非常的不可靠，有这么一种情况，很多人稍微比平常练多了会，就觉得好累，受不了，不行了，感觉已经力竭了，可是这时候如果有一个健身教练在旁边，经过指导的话居然很容易再做一组训练，你觉得你早就不行的时候，居然能够在做一组，这不神奇，也不是身体的潜能，这只是没有练没到位的结果。

以前我也是觉得一边跑步，一边听歌，以为转移注意力就可以跑得更轻松一些，错了，这个真的打错特错，感官刺激也会消耗注意力，可能的问题是注意力消散了，身体其实才刚热身。而这就不能让身体达到“力竭”状态，而没有达到这个力竭的锻炼会打很大的折扣的。这个一定要谨记，力竭很重要，力竭很重要，力竭很重要，重要的事情说三遍。抗阻训练时，肌肉是否训练到位的标志就是力竭简单来说，每组动作做到不能再做下一个，就是力竭的表现，这说明肌肉的供能系统已经消耗完能量，需要休息重新储能。

运动达到有效心率时的感受：猛烈而急促地喘息，强烈需要氧气

抗阻训练组间休息时间控制在30s-90s最合适，这个时间内ATP和磷酸需要再合成，太短疲劳未消除，太长神经兴奋容易消退，很难获得泵感，

这里再强调一下，要全身心的投入，其实做什么事情要全身心投入效率都会很高，健身也不例外，可是好多人都有类似我以前的想法，一边健身一边娱乐，顺带把一部美剧也追完了，要知道这是那些已经非常有自制力的人的表现。全神贯注的做一件事的意思是，要把所有的注意力都放在呼吸和现在做的动作上，这样才能够清晰的感受力竭这种感觉。而真正的进入这个状态的时候，你会发现你根本听不进去音乐，看不进去视频，因为你的脑袋已经完全被占据了。你也更可能完成最后几个动作。（我刚才说了呼吸，呼吸这个我又一个技巧，就是尽量的让身体感觉到，尽量“大声”呼吸）

吃的对

不论是增肌和减肥其实都要有一个目的，那就是增肌，一些经典的健身书籍，都会扯一个道理“三分练，七分吃”，可见吃在锻炼里的重要性，知道我们的目的，那么其实就应该尽可能多的摄入蛋白质。网络上有很多的健身餐，大家可以参考自己做着吃，这里不多介绍，多吃牛肉，鸡胸肉，鸡蛋白等高蛋白食物，其它的就顺其自然吧。
