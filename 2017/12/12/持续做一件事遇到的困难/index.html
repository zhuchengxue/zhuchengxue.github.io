持续练习必然遇到的困难

发表于 2017-12-12 | | 本文总阅读量 次

要掌握一项技能，比如写作、英语、健身等，必须要做到持续练习。

而我们在练习的时候总是容易低估可能遇到的困难，所以才导致很多时候开始激昂奋起，到最后无疾而终。

我也看过太多的文章和书籍，但感觉都不适用，适合自己的才能持续，根据自己的经验，分享三点真正的干货给大家：

生理性疲劳

当我们在做练习的时候，某一个器官会疲劳，这是不可避免的。

比如我们做运动，跑步久了自然腿就会酸痛，

长时间对着电脑办公，眼睛自然会肿胀，因为长时间近距离的看屏幕，加之屏幕是会闪烁的。怎样放松这个疲劳的器官，使之能够更好的休息。

我一直在寻找解决某个器官疲劳的方法，后来在健身的过程中学习到一个方法

一周5练的人，他们锻炼的策略是，每天锻炼不同的肌肉群，当天不训炼的肌肉就自然在休息了。

这给我很大的启示，其实“不用”自然就在休息了，所以调整的方法就很简单了，把注意力转移到其它的感官（味觉，触觉，听觉，嗅觉）或注意力直接停机（睡觉）。

我们可以做的是什么？比如眼睛疲劳的时候，那么我们把注意力从看这个感官动作移开，我们可以冥想（注意力在呼吸）、吃一个喜欢吃的零食（味觉）、听一首动听的音乐（听觉）、按摩（触觉）、运动等。

心理情绪波动

当我们做练习（practice）的时候，没有情绪波动是不可能的。

在自我救赎过程中，最最大的阻碍其实是自己的情绪。比如说你在学习的时候脑子里不间断的冒出，别人都在玩耍打击你为什么还一个人在这边学习或者为什么以前不好好学习现在还要苦伶伶的一个人在这边学习。再或者。因为进步好小，这样学习还有意义吗？是不是很浪费时间呢？还该不该继续学习呢。还有没有其他方法可以改进自己的学习方式呢？然后又去找其他的方法，然后没有在一条路上走到底。这真的是。非常磨砺自己，阻碍自己。最大的困难吧，应该说。

因为那种练习很久可就是进步不大的沮丧感时不时会涌上心头，很容易想要放弃，脑袋里有一种声音“可能自己真的搞不定吧，要不算了吧……”

怀疑的念头一起，接踵而至的就是没劲、灰心……

墨菲定律告诉我们，越担心什么事，那么这件事发生的概率就越大

后来，真的就算了……

情绪在持续练习中的损耗看似很小，但影响极大，个人体会极大。

你看我们总是在那些没有必要的事情坚持到底。科技的发展让我们在也不会无聊了，少年人有追不完的漫画，刷不完的小红点，莫名其妙的哀愁，反正就是激素一分泌，心情就变样。

比如习惯性的拿起手机解锁，打开微信滑动几下，无人回应，点滴失落，漠然锁屏，隔上不到五分钟，又要重复上面的动作。对于这个恶习，最有效的办法，只有记录，因为即时反馈对自己的大脑就是最大的奖赏，不可能一次性就可以矫正完全，当看一次手机就记录一次，当一天下来，自己在刷手机这件事上花费了这么多的时间的时候，我们就会知道自己做了一件多么愚蠢的事情了，也就能够放下这份偏执了

如果因为某件事情，可能是家里的一些琐事，人情之间的膈应，导致心情低落，那么基本上是做不了什么正事的，因为注意力根本就集中不起来。

除了尽量避免让自己陷入一种过度的情绪，保持内心平静外，心态上，真的得要自己给自己鼓劲，这很鸡汤，不可否认的一个事实是，我们的生活离不开鸡汤，在心无头绪的时候，顺着鸡汤的思路，也确实是一种门路。

这时候要相信自己，时刻提醒并暗示自己总会变好的，时刻留意自己练习的状态，这个阶段必须经历，放下无谓的怀疑、踏实认真的做好功课，完成每天的练习，直到奇点的到来。

关于拖延

管理学上有个词叫做“学生综合症”

指的是人们像学生们写作业一样，总是要拖到截止时间快到的时候才开始努力工作），这会导致当项目发生偏差或出现问题时候，前面为项目预留的安全时间白白浪费了，即使项目进展顺利，也只是按时完成。

学生综合症类似的还有个帕金森定律

人们总是把工作拖延到它所能够允许最迟完成的那一天。如果时间充裕，且允许推迟的话，人们会不自觉的放慢工作节奏或拓展工作内容，很少有提前完成的。比如一天的工作，其实可能半天就可以做完的，但很多人要做一天才完成，可能有这么个顾虑，避免承担额外的工作

因为持续的练习都是为自己，但大脑有个特性，能不用就不用 ，它本身就不想思考的。如果可先玩下游戏，看个片，自然而然把重要的事情落下了。

人性的弱点，要克服非常困难，认识到至少可以时刻提醒自己。而不至于陷入太深而不自知。
