三十随想

发表于 2017-07-05 | | 本文总阅读量 次

年龄

年龄在中国是一个非常玄的东西，它分为实岁和虚岁。一般的中国人都是按照实岁来算的，因为这是身份证上面的岁数，在我老家那里都是算虚岁。而真正意义上的生命的开始应该是虚岁，因为这个是受精卵生成的日子，也就是真正意义的生命出现的时候。

这么多年来，岁月如水，恍若不闻。悄悄然，三十载年华已过。回首望，是如登高往下看，太下面的风景和人都变得非常的微小并模糊，使劲揉眼睛想看清，但已经不可能，终究是渐行渐远了……

意识

佛教有“十识”的说法：

说的是除了“五识”——眼识、耳识、鼻识、舌识、身识外，还有意识、末那识、阿赖耶识、阿摩罗识、无量识。

当然我认为最重要的其实就是第六感即“意识”至于后面的几个感觉太过深奥，或许要看造化才能碰触吧。

对于生日的意识，小时候就是和爷爷一起过，我和爷爷相隔一天，我12他13所以家里人就我们一起在13日庆生，顺便把我生日过了。太深刻的印象确实也没有，只是记得小的时候念想过生日能吃到好吃的，其它没了，倒是到了二三十岁的时候，每一年的生日都在提醒“你又老一岁啦”。

因为在中国这个特殊的国情里，人们其实并不想长大，因为几乎所有条件不是很好的人都一直在准备着“翻身”，但一直也没有准备好，所以害怕长大，不想长大，担心忧虑，长大就要结婚就要买房，就要生娃。至于生日这事相比这些根本就不重要了。过生日唯一的意义就是提醒你又老了一岁了为什么还没娶老婆/嫁汉子……所以有好一段时间，应该也有十多年了，我是非常的排斥生日这个话题的。

人们潜意识里觉得或许需要用庆祝的方式来对抗时间洪流，或用狂欢来忘却总要到来的节点。但其实没有用，该慌张的照样慌张，该茫然的还不是一样茫然……

网络上有一个关于巨婴的说法，巨婴的主要特征如下：

  1. 无法观察到事物变化的趋势，只有在自己亲身感受到之后才恍然大悟，那里知道已经是传导末梢了。
  2. 因为只有短时记忆，无法驾驭一个比较长的观察时间单位，只能以天，最多为周；无法以半年、一年的单位来看待趋势。
  3. 即使吃过亏，即使是一种循环，也没有建立自我的反省机制，导致同样的坑反复跳。

记录

开启了反省意识，其实也就是现在人造名词“元认知”等，一有感官意识，我们很容易想到，时间虽然不留痕迹，但意识犹在，只要有记录相关的那时岁月的一点特征，其实人也是很容易在脑海里穿越时空，而且幻想根本傻傻分不清，儿时的感觉自然流淌。而这对于个人来说明显是非常的重要的。但我认识到这一点是非常晚的，属于后知后觉那种类型，也只是证明了我确实没有很大的天赋。

更有甚者，我现在已经基本上不相信自己的记忆了，因为即使是上周或者上个月发生过的或者做过的事情，现在的我已经基本没有太大的印象了，但如果我做了记录，就很容易回想起来。

我开始相信记录的力量。时间就是这样，如果平常自己关注的都是一些不关我事的事。而关于自己的信息其实又寥寥无几，基本上就毫无知觉的过去了。我相信记录这件事是对的事。

正如去年醒悟的一件事

做对的事情远比把事情做好更重要

对于我自己，这也是今年最大的收获之一。

真的到来

乃至于终于到了而立之年，我开始并不是那么害怕年纪的增长了，因为:

  • 怕或不怕，这都会到来，也终将过去
  • 没有得到，也算不上失去
  • 世人如果没有发明“年”这个周期，这其实就是非常普通的一天
  • 顺着趋势，其实就是开始40,50,60……人生如草木枯荣，整个也就是周期性的生长，这就是自然
  • 无信仰，潜意识里确实也没有神鬼的概念，但开始相信一些东西，比如真，比如刚说的记录等

最后

祝愿自己在未来的这一天，不需狼狈，不需纠结，岁月如梭，做不到闲庭信步，也能够无愧我心。
