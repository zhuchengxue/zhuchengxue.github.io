锻炼的一点经验

发表于 2017-07-03 | | 本文总阅读量 次

开始

我健身的动机一开始就很简单，我想要的是更有精力和活力，因为属于上班一族，需要每天呆在电脑的，呆的时间太长了，又是久坐，每到午后就会非常疲惫困倦，虽然午休是一个好提议，但是公司不能午休······

而我也注意到有些人，而且比如小孩子精力旺盛的很，根本就不需要午休，确实只能接受身体不再年轻了？

每个人刚开始准备锻炼动机，男人可能是肚子，女人可能是裙子；这只是一个念想而已，因为这种状态持续不了太久，这就类似高潮也是就那么一下下，引起脑袋兴奋，激素褪去 的时候，慢慢恢复正常人状态，该吃吃该喝喝。

既然意识到是自己的身体退化了，总想补救补救，该好好去锻炼身体了。

可能是误区

健身的方法太多了，现在网络如此发达，随便一搜，什么囚徒健身啊，硬派健身，街头健身，州长训练百科全书啥的好多，刚入此门中，置身于这么多教程，有种自己也终会是其中一员的幻觉，相信自己好好练，早晚一天也是下一个施瓦辛格，至于到现在为止，还有一些女生还以为，我练器械怕长好大肌肉的人，人啊总是有一种天生的自以为是，自以为是到自己都没意识到因为自己的无知才会提出这么傻逼的常识性问题。

很多没去过健身房的人，看到那么多器械都，好是兴奋，前几次练得不亦乐乎，可新鲜感过后，就那样和平常跑步啥的一个样，累的狗一样，虽然也会经常去，但因为进步实在不容易，倦怠心情不可避免，急于求成的心态油然而生，而这又太容易引起恶性循环，最终导致办了年卡会员，到最后一年过去了就只去了前面的两个月，而这还是有意志力的人。

人都是好聪明的，看到那些肌肉块头大的在练什么，等他们练完之后，也练这个，但力不从心，但既然这个重量搞不定，那就换轻点······可这还是没有啥用，因为他们不知道该练到什么程度 ，可他们又会想当然的以为自己做的虽然不是很好但估计也不是太差。

以前跑步的时候，那些有经验的人说要至少连续跑30min以上有氧运动才有减脂效果，所以好多人也真的这样跑了，非常少的人由于意志力确实很强，每次坚持了一个多月，瘦的明显，可这太过难了，一直跑啊跑，跑伤膝盖了却得不偿失，普通人只为健康而不是锻炼肌肉何必。

该怎么锻炼呢？

比如跑步，解决方法是慢点跑，这样才能够持久，慢点才能够有力气跑完。但一个前提是在自己的耐力每次+1训练，慢点就不会像那些用力过猛的人，练了一次就要休息了好多天的，第一天消耗的卡路里确实远超所摄入的卡路里量，但休息几天后终于“恢复”过来，而且恢复过头，胖了。

现在选择健身亦或是阅读的书目的时候，我很在意这本书是否体现循序渐进这种机制，而这在健身尤为明显。凡事都有一个过程，而这个过程中在哪一步要进阶，都是有提示，身体要达到怎样的地步才能够进行下一步。

要知道人和人是完全不同，高矮胖瘦如此泾渭分明，真心不能一套方法走到底，这里又涉及到因人而异了，而这又需要自己去慢慢研究探索自己的节奏。

一点经验

以前高中学过一篇古文叫曹刿论战，一鼓作气，再而衰，三而竭，这个在心态也可以接受，比如第一组的时候我很有劲，做完一组感觉还行，休息到呼吸均匀，好再来一组，这次有点吃力了，但还是完成了，休息，终于到最后一次，心里想着事不过三，练不过三，做完这组就好了，心里有准备，虽然也是很累，但还是使劲做完，气喘吁吁，好想趴。不过做完了不一定会达到力竭，而练到力竭状态才是真正有效的。

怎样才能做到力竭呢？

小时候我很喜欢看别人打街机（暴露年龄的游戏）里面有个格斗游戏 叫“拳皇”，我最喜欢看那些高手可以连招干翻对手，这本来就是大招，几个连续的大招下来，对手的血基本降到一半以上，太过牛逼，所以用到锻炼这方面，我是怎么做的呢：单调的一组相同动作重复做四组这是很多健身书籍建议的做法，我想把连招也带到锻炼上来，比如刚练完腹直肌上束肌肉群，立马做深蹲一组，做完在俯卧登山一组，完整下来就是一个连招模式，如果还觉得ok可以再加一组腹直肌下束训练······这样下来一般就累得够呛，休息到呼吸均匀在来两组，练不过三，效果呢，你试一下就知道了。

在练的过程中，中国武术里有一个词，叫念动合一，这里有个自我暗示的效果，相信什么就真的会发生什么。

想成长就不能做那些看起来毫不费力的动作，锻炼身体就是让痛苦和极限不停地刺激自己，通过不断做你以前做不到的事情，才能进步，而这个过程真的需要慢慢来，不能急。

身体真的需要张弛有度，才能长久。

执念

最后，锻炼必须要有执念，这个可以自然让人如例行公事般坚持下去。总是要做的，不能不做的······如果能做到痴，那就早晚功有所成。
