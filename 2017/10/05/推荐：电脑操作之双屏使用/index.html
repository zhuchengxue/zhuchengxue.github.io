推荐：电脑操作之双屏使用

发表于 2017-10-05 | | 本文总阅读量 次

为什么使用双屏

其实现在很多人家里都有两台以上的电脑，但同时使用两台电脑的人并不多，同时使用两个屏幕的少之又少。

同时使用两台电脑，需要两套键盘和鼠标，而频繁的切换电脑鼠标和键盘，不单不方便，而且没地方放啊。而且如果要在两台电脑之间实现复制黏贴，或者移动等很常规的操作，非常困难（也可以实现，通过一些例如dropbox，或者共享鼠标的软件等）那种滋味我试过，真的心焦。

双显示器就不一样了，这其实还是一个电脑，只是把原来的电脑屏幕扩展出来，增大他们的边界，而这又相当于两台电脑，因为有两个屏幕嘛。两个屏幕间，鼠标的移动如丝般润滑，完全感受不到停滞感；那种感觉就像从一台电脑换到了另外一台电脑，而且是无缝连接；这种感觉非常的棒。

在我看来，其实双屏的精髓在于：

把本来堆叠起来的应用或文件，平铺开来，让人一目了然。

使用双屏场景

  1. 经常做文字处理工作的人，或者说使用office或其它字处理软件的人，一定会由于屏幕太小，或者打开多个文件，应用间的频繁切换找个文件翻来覆去的经历痛彻心扉吧。复制个短语都要花好长时间。有了两个屏幕查看，复制，如此自然了；
  2. 阅读原版书籍或文献的时候，一个屏幕打开原版，一个屏幕打开翻译或者浏览器查询或打开记事本记录关键摘要；
  3. 学习编程的时候，一个屏幕代码示例，一个屏幕IDE
  4. 写作的时候，比如使用typora，在其中一个屏幕全屏，沉浸氏写作，没有干扰，另一个屏幕可以查询素材，搜索等需要的时候快捷键切换一下就好；
  5. 看片的时候，一个屏幕全屏看片，另一个屏幕可以看影评，或浏览微博twitter等，不影响；
  6. 学习英语的时候，一个屏幕全屏教程，另一个屏幕全屏有道词典或mdict等词典随时查询；
  7. 一些需要对比的场景，比如校对文稿，比如横向比对文献；
  8. 整理资料的时候，简直不能更简单，之前一直想要学习使用total command，现在发现完全没有必要了

以上只是我自己使用的过程中，显著提高效率的一些场景，更多精彩需要个人自己发现。反正就是之前一个显示器不方便或不能同时进行的事情，有了两个显示器就可以轻松实现了。

双屏使用技巧

下载一个displayfusion的软件，这是一个专门为多屏软件设计的，很喜欢里面的一个功能非常棒：就是它会在右上角最小化图标旁边增加一个功能键，点击或者使用快捷键（ctrl+win+x）就可以把当前窗口瞬间移动到另一个显示器

我崇尚使用快捷键，因为快捷键可以极大的提高操作效率，虽然现在并不能够完全只用键盘操作，但80%的操作都可以使用键盘，这里举例几个常用的快捷键：

  1. win + 数字，任务栏可以固定一些常用的软件，比如浏览器，文件夹，同时按住win和相对应的任务栏的位置就可以快捷打开相应的软件；
  2. Alt + tab ：切换不同软件
  3. Alt + F4,：关闭当前窗口，因为两个屏幕虽然没有延迟但距离却是变长了，这个快捷键直接关闭；
  4. 稍微学点autohotkey，然后自定义一些快捷键（引用）

其它的快捷键不必太在意，因为本来就是一台电脑，就相当于在一个屏幕上操作一个样的，所有的快捷键都和一台电脑完全一样。熟悉习惯后，就会发现这真的是节省时间的利器。

如何安装双屏幕

  1. 确认电脑可以外接的显示器接口，一般有HDMI、VGA、Display口，找到相应的转接头；
  2. 两个显示器（最好是大屏幕比如24寸以上）
  3. 一个显示器可以竖起来，一个横着放，连接上电脑，快捷键（ #+p），就可以设置扩展屏幕了。

最后

自从开启了双屏，我的宅生活又到了一个新的境界，这是令人欣喜的事情。

一直想要拥双屏，但一直犹豫要不要下手，主要担心自己可能并不是刚需，然后怕自己以后挪窝的时候不方便，还有个原因主要是转接口这个不懂怎么搞，不知道可行不可行……幸亏有爽哥的帮忙，原来真的可以的。

当有了两个显示器后，感觉自己之前为什么不早点下手啊，之前的顾忌真不该有的。想方设法都应该搞到哈。

知道一个东西很好，也经过了别人的验证确定这是很好的，那么正确的做法就是应该在有限的能力下尽可能的满足自己的需求，只要可以提高效率的事情和物品，花多大代价都是值得的，提高效率其实就是延长自己的寿命。

这应该是常识，也是作为以后生活应该具有的最基本的认知。
