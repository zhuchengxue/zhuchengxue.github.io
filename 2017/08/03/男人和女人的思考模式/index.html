男人和女人的思考模式

发表于 2017-08-03 | | 本文总阅读量 次

我们看书的时候，总希望能看出一点不一样跟东西出来或者学到一些技能能应用到我们的生活里才不枉费那么费劲的看了那么久的书。书里面的内容是把这些经验平铺成文字印在了纸张上了，我们需要吸收这个经验，然后内化成我们的技能，这才是读书的最高境界。

但就像一千个人眼里有一千个哈姆雷特，即使是同一本书，不同的人也会产生非常不同的感觉。但不管怎样，我们总是希望能在那些大众认可的书籍里寻找我们潜意识里渴望的那些真知、快乐、美好，或者和女人相处的技能。

细心思考一下，当我听到媳妇抱怨（很大可能关于家里的琐事）的时候，我的第一反应就是到底哪里出错了，是我关心不够？是她又犯傻了？还是公婆对她做了什么大不了的事情？亦或是纯粹是“大姨妈”来了…… 如果心情还好的时候，我还可以耐心的听她抱怨会，耐心和她讲道理，给他分析轻重缓急，善恶利弊等等，心情不好的时候，不管了，女人发疯简直不可理喻，这样反而会火上浇油，她变得更加躁狂了，一发不可收拾了，最终我被打败了；

但其实起初的原因并不是我造成的，为何最终我却成为了爆炸的唯一“受害者”，可怜一个无辜的人遍体鳞伤。我承受了那恶毒刺耳的抱怨负能量，如果一而再再而三，不用说，人会崩溃。

当情绪冷却下来的时候，我认真反思过好多次，个人自认为是一个极为有耐心的人，可是怎么也会乱了分寸，爆了脾气了。情绪爆发的那个阙值我是否可以感应到呢。

仔细回想我自己情绪失控的几次意外，那时候是是有症状的—双方的安全感到达了一个极低值。

想想是这样的：

男方是经过一天的工作，他认为女方居然一点都不体谅自己，每天就只会在鸡毛蒜皮的事情上挑刺，他会觉得别人家的老婆不会这样的……

女方觉得自己一个孤弱女子，嫁到了一个全部生人的家庭，老公一直在外面，也没有时间陪伴自己，天天面对的是他老娘，总觉得她看自己不顺眼，又听到了什么风声，想想自己的闺蜜家庭，她的老公，心里有苦……好不容易等老公下班，想和他聊会体己话，然而他总是那副“尿性”，不闻不问……

最终闹到不可开交，不慌而散。和女人打交道一定是有技巧的，因为不可否认有很多非常和谐的夫妻或情侣关系，这里面关于常见的套路话术，也一定是有的。

看《魔鬼约会学》当魔鬼老师书中举例的那些巧妙的回答，设身处地的一想，真的非常妙，当然我的目的不是为了泡妞，我是想在那种在我看来可能紧张得要死的情景里，比如搭讪，TED有一个视频讲得是人一紧张的时候，大脑就会分泌一种叫做“皮质素”的物质，让人反应迟钝，就是你那种你木讷的时刻。有些非常内向的人，还会感觉手心额头冒汗，脑袋一片空白，根本不知该怎么回答，可能心里就一个回音“我要完了，我完了……”他是怎么可能想到那么妙的回答呢？这背后一定存在一种智慧或学问。

我也确实发现了很有意思的东西，不同语言模式：

  • 作为适合应激状态的男性思维，其语言模式具有如下特征：上堆、抽象概括、做出评价、解决问题、探求动机、预测未来、建立因果、推理总结、防御否定、目的优先、结果导向、以客观为标准
  • 作为适合自然状态的女性思维，其语言模式具有如下特征：下切、具体细节、表达感受、描述问题、关注过去、对照眼前、罗列现象、就事论事、开放肯定、忘却目的、活在当下、以自我为标准

当然对这两种思维模式，还有更深入的概括，所谓男性思维一般是面向未来，而女性思维是面向过去或现在的。一点小技巧，当要表现友好，主语要以你开头。

这是不同的思维的趋势特征，想要聊天就用女性思维，不想聊天或者想要解决问题就用男性思维，而男性思维是应激情况下的反应，而女性思维是自然状态下的反应。这样看来形容男女吵架其实就是“打仗”这是非常符合的。

当然一直处于应激状态下，人容易崩溃，而处于自然状态下，人会觉得幸福。

而我们其实这一辈子或许很多人就是只为了获得这种幸福感。那么关键来了，其实我们就应该努力趋同自然状态 ，自然状态其实就是你真实的状态，也就是说最容易保持的就是你真实的状态，因为所谓“装”，总是要用一个谎言来掩盖另一个谎言，最终注定会露出马脚的。

最后，送上书里面非常接近“道”的话：

古往今来，那些智慧的人往往愿意以无常的态度来看待生活的起起落落，他们会看中事实，会就事论事，但不会轻易去建立因果关系，这样也就控制住了自己的情绪，不会大喜大悲。反之，那些脆弱的人在被情绪左右的时候，甚至还会主动去寻找相应的事实，结果不是进一步作践自己就是死磕别人
