真正能改变生活是做事

发表于 2018-03-07 | | 本文总阅读量 次

其实我清晰且正确的知道，只有行动才能改变生活：

  • 想要减肥，跑起来；
  • 想要赚钱，去工作；
  • 想要成长，去学习；

但是这样想的话，太过简单，一开始实行，就马上遇到困难。

工作找啥的好？现在学习学什么？还来得及吗？跑步或许真的能减肥，可以自己的意志力，估计也坚持不了多久……

当人们陷入困境时候，时时刻刻希望能够速成，做什么事情都希望能获得点什么。

考虑事情的思路一般这样的，也许这就是个机会呢？希望不要太难，其实只要是个机会，我努力是没问题的。能力这种事情，确定是机会再练。然后不断的纠结：

  • 编程这个技能值不值得学习，以后能不能赚钱；
  • 去大城市那么远的地方，有没有好的工作，如果去到那边，最后还是现在这个样子，还不如不去；
  • 学英语啊，没两天，开始抱怨英语太难了，我就是学不会啊！

不断的犹豫，然后又开始怀疑人生，到底做什么事情，才有钱途，不断的陷入困境。

生活完全处于失控状态，一切都是被迫的，感慨人生不由己。

正向循环

失控的人生，最好的方法就是马上建立正向循环。

古典在《跃迁》这本书提到了几个正向循环：

  1. 持续学习，是“学习—成长—增值—学习”的循环，这个循环的关键点在于学习能“增值”
  2. 坚持锻炼，是“锻炼—精力充沛—锻炼”的循环，关键在于锻炼是否能“精力充沛”，而不是“训练过度”
  3. 持续帮助他人，是“善意—回报—善意”的循环；那种“输血式”的助人，不是真的想帮人，只是求认同罢了。

当然还有其它的正向循环，比如企业的“信—认”循环、兴趣培养到能力养成循环……尽量寻找并且跑通正向循环是每一个追求幸福人生的人必须思考的问题。

死亡螺旋

有正向循环那必然有恶性循环。

恶性循环类似重力，不断的把你往地狱深处拉的力量，有个名词形容得很确切，把这叫做“死亡螺旋”

四个死亡螺旋要警惕：

  1. 穷者越穷：忙—乱—忙，稀缺—没有资源想长远的事—稀缺，物质匮乏—诱惑多—自控力消耗—物质匮乏
  2. 投入不足：很多企业面临这样的问题：利润下降导致研发投入不够，研发投入不足让竞争力进一步下降，与此同时公司还不敢停，因为养着很多工人和厂房，一旦停大家都得挨饿，就像坐在火山口上，惶惶不可终日；
  3. 工作狂循环：工作狂—家庭投入少—家庭没成就感—工作狂
  4. 做自己：求认同—找不到自己—求认同。越是求认同，注意力就越放在外界的关注、外界的高手上，也就越难找到自己的天赋、梦想和激情，这也让他们越来越焦虑。

高手做事，就是能快速识局，然后努力搭建正向循环系统，破坏负向死亡螺旋，切断自毁线路。

高手的招式

学习一些能自增长的技能
有一些技能天生自带自增长能力，非常重要。

  • 读写能力：好的读写能力能让你接触到更多好资料，成为更好的读者或写手。
  • 破英语：我不认为成年人需要把英语学到多好，更聪明的做法是够用就好的“破英语”，熟练利用谷歌翻译、维基百科、亚马逊、字典软件、搜索网站迅速找到大量英语资料，在应用的基础上慢慢提升。
  • 社交能力：人际交往能力会让你认识更多人，反过来强化你的社交能力。
  • 解决问题：尤其是在系统的层面，你看到的系统越多，世界对于你而言就越简单，于是你就有更多资源来理解世界。你慢慢会看透世界的规律，实现思考力的跃迁。
  • 聪明的善良：善良是世界上传递最远、增长最快的东西，但是笨拙的善良往往会带来摧毁你的负循环。聪明的善良是重要的自增长技能

路漫漫其修远兮，吾与大家共勉！
