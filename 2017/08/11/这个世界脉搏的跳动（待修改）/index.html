这个世界脉搏的跳动

发表于 2017-08-11 | | 本文总阅读量 次

所谓的世界

诚可见，我们知道我们生活在一个信息非常发达的世界，可是真实的情况是：

几乎所有人都只是生活在一个小的空间里，在这个非常小的空间里，一日三餐，朝九晚五或起早贪黑，上学放学，这个空间可能是一座小区，小城，村镇等。能看到的也就是熟悉的亲人朋友，路边的花草，和昨天的也没有什么太大的区别，亲身所见的空间其实就是自己所处的小世界，我比较喜欢称谓空间。

刚说了空间之外苍穹之下，我们的世界是一个地球，通讯的发达，一个手机就可以连接外部的世界，要知道在以前的世界里，信息的流通只能通过广播，报刊，书籍，公告等，互联网的发达，手指尖就可以轻松接触世界的信息，只要你想。一个小空间通过协作，达到生活的和谐。

在深入本质人所认知的世界，是自己感觉的世界，有意识形态的世界，也是可以生根发芽的。在这个世界里，各个领域比如（物理，数学，英语，文学，音乐……）不经意的打开，由于自身的积累，每个人看到的景色和人文都是不一样的。这样比较的话，那些内心大世界的人，他们的内心是非常的丰富多彩的，而少见识或者都没有读书的人内心的世界是苍白空洞昏暗的。

每一个世界都是一个活的生态系统，维持它的运转，是能够形成一定的循环的。大千世界，无奇不有，乱花渐欲迷人眼，东瞧细看，走马观花，也就走个过场而已，这没有什么，也能过，但要把握这个世界的脉动，或跟上世界的节奏，我们要学会聆听这个世界的声音，脉搏跳动的声音。

能听到世界跳动的声音的人，就是能够把握住时代脉搏的人，不然也就只能停留在成为这个世界的小细胞里面而已。

因为是信息化的世界，所以很多信息并不是知识，相反这还是噪音，会妨碍你听到想听到的声音。

世界“跳动”的特征

有声音

当地震来临的时候，天崩地裂，当风雨欲来的时候，乌云密布；当世界跳动的时候，他会山呼海啸的奔腾而来，事后，人们会把变天叫做“革命”，处在浪潮之巅的人，引领潮流，顺风而起的猪，青云直上。

有节奏

历史是有周期的，随着时间的滚滚流逝，因为人的健忘，人们会一直放相同的错的，本质上这些错误都是相同，周期的规律性，让人感叹这个世界的有高潮有低谷，如人的情绪一样。

新领域

闯入新的领域，就像在自己的世界里开荒拓土，进而建立起属于自己的高楼大厦。这是属于勇士的荣誉，需要突破自己的安全边际，单刀深入太深，又容易产生焦虑害怕和怀疑。

元认知的神奇

第一次听到，我们是可以思考我们怎么思考这件事的时候，我就觉得，大脑好像监控一样，可以监视我们的行为和思考的一举一动使之有知有畏，又如自己世界的上帝一般，时刻指引着自己向前。

那些能够触摸到这个时代脉搏的人，都是一些非常成功的人士，他们的见解能够清晰的描述这个世界的运转，而我们只是普通的听都能够受益匪浅。

普通人的机会

多看书

只看自己真的相信有用的书，任何人的介绍都不能逾越这个准则，要看就认真看，而不是说只是为了快点看完，不然很容易有偏见，只是看了之际想看到的东西。

不看书，但不停止思考

思考，应该是一个大脑皮层发育完善的物种应该必备的技能，但自以为是的认为自己是对的，也是一种短视，一页障目而已。遇到问题，我们要考虑发生问题的原因，而不是等着这个问题成为犯错的原因。

对趋势的判断和洞察力

这个比快速学习和努力勤奋等重要得太多了。
