该不该读书

发表于 2017-08-07 | | 本文总阅读量 次

我的一个兴趣爱好就是读书，听起来像一个书呆子，对很多人来说，读书是一件很累人的事情，可是我如果几天没有读书心里就会很愧疚，感觉浪费了好多的时间，然后就会到处挤出时间来阅读。看书对于我来说，其实就像别人喜欢打球，喜欢游戏一样，这是一种娱乐活动，只是这是主要是娱乐自己的脑袋而已。

看了《原则》之后，我经常会把里面的一个原则以我的方式运用起来：

凡事想得深一点，想想做这件事的，一级效应、二级效应、三级效应……

就像下棋的时候，多想那么几步，这样子就比较能够周全的看待事情。

好，我们考虑一下看书这件事情，读书的一级效应，可能有两种情况：

  1. 喜欢读书的，看书就像吃饭一样，到点该吃，拿到书就读，并没有什么特别；
  2. 不喜欢读书的，看书如受罪，就像不喜欢锻炼的人，与其自虐，不如睡觉。

读书的二级效应：

  1. 喜欢读书的，因为读书，学习到了一些知识技能，开阔了眼界，提高了自己，良性循环，更爱读书了；
  2. 喜欢读书，但读的书很难，一本原版读了好几年，可是还是在第二页徘徊，开始怀疑自己喜欢读书，或者只是喜欢那些简单的书籍，脑袋不够？开始怀疑自己是不是真的喜欢读书了。
  3. 不喜欢读书的，但知道读书是有必要的，所以硬逼着自己读书，看书很累，文字简直就是催眠符；这是做正确且有必要的事情，但缺的是兴趣。
  4. 不喜欢读书，但别人逼着读（家长、老师），想方设法跟他们作对，能不读就不读，不读就赚到了；

读书的三级效应：

  1. 而读了好多书的你也确实感觉到“知道那么多道理却还是过不好这一生，而且有那么多书，穷其一生也不可能读完。自然而然会想到比读书更重要的事情是什么？比如说思考或学以致用，内化行动等等，读了大半辈子书，居然脑袋越读越乱了；
  2. 读书加上思考，不仅学到了还做到了，读着读着就成了行走的艺术了
  3. 不读书的如果生活过得去的，感觉自己选对了路子，因为读书没有用嘛，那个读书最厉害的呆子现在都还没房没车，更别说老婆了，心里自得其乐
  4. 不读书，心智未开，而且确实吃了没读书的亏的，自己读不了，一定要让自己的孩子好好读书；

我从一开始划分两种人，喜欢和不喜欢读书的，然后在上学阶段，再到长大阶段，对读书可能产生的后果进行不完全的预判。

读书会产生焦虑，有太多的知识，所以越是学识渊博的人，越觉得自己懂得太少，反而是没读书的，活的淳朴善良；

一直学习，或者说现在看到的终身学习者，学习上瘾者，好像我就是这种人，每天总想有些新的认知，新的人生感悟，新的角度看为题，新的思考方式，反正总是有得学。所有的时间都放在自我的“充电上”，好像这样就能够避免被时代所淘汰……

但我也算看了点书，可是自己的生活并没有很大的改变，至少对我来说，读书这件事出了点问题。

有人说是看完了你没有行动呗，只是脑袋爽了一下，就像高潮那么一下下，过后，没几天就又忘了，读书一读就忘，就觉得更应该读书

直到看到世界最著名的科学家爱因斯坦居然怂恿说不要读那么多书，他的理由是太多书会让人变得懒惰，不用思考，以为别人都帮你想好了，所谓的喜欢读书，其实就是想做一个高级的“伸手党”，归根结底，其实就是脑袋不想动。

当然脑袋就是这个德行“能不思考就不思考，即使需要思考，也不思考”

或许我已经知道了问题所在了：

读书是为了促进思考

不要陷入为了读书而读书这个怪圈。

最后还有一个问题：那读书读到什么地步就可以不用追求读书了？

到那时候，我对我想了解的领域已经足够深度了，也能够解决这个领域可能遇到的几乎所有的问题了；对于自己的思考也建立了一整套完整的体系，跟写书的人，即使面对面也可以谈笑风生了……或许就不用读了吧

不过我觉得需要在多读至少100本书，可能才能达到这个境界吧。
