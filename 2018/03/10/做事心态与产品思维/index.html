做事心态与产品思维

发表于 2018-03-10 | | 本文总阅读量 次

前天，一位朋友推荐给我看了3篇梁宁《产品思维30讲》，有点意思。

网上查了下，江湖上传言她是“中关村第一才女”……

花两天的时间把30讲全部看完，里面讲了很多的互联网的思维，确实有料，这里讲两点非常有意思的观点：

####1、 五个层面识人

一流产品经理有一套的方法来识别一个人：

  1. 感知层，这好理解，也就是以貌取人，对方的身材相貌、衣着打扮基本上第一印象就是我们的感知层；
  2. 角色框架层：所谓人在江湖，我们都在角色扮演，一个人的角色属性，比如白领、公务员、领导等，角色的属性决定了日常沟通交流的套路；
  3. 资源结构层：这里包含了一个人的财富资源、人脉资源、精神资源，随着成长每一个人都不一样。
  4. 能力层：一个人能力越强，对于自己的角色就驾驭得越好，进而可以发展自己的事业；
  5. 存在感： 这是一个人的内核，以我的理解这个就是一个人为什么这样子生活的原因。

举个例子，几乎每一个人，都有几个有钱或有权的亲戚，谈起来他们来，感觉就是自己的荣耀，可是细究他们之间的关系，也就是点头问候一声而已。

而点头之交的缘由，绝对不仅仅是由财富和权利造成的。

又或者，如果想和人建立深层次的关系，比如婚姻，单看一个人外貌怎么样、衣服穿搭，或者只了解一些非常表面化的资源，比如有多少存款，有没房子车子等，那是完全不够的，必须从后面三层深入了解。

以作者的观点，一个人为什么奔波，为什么不安，为什么焦虑，机关算尽，一直驱动男女思前想后、废寝忘食、找人找事找钱，去满足自己的，其实是存在感在作祟。

因为如果一个人的存在感满足了，那么ta就不折腾了，就好像好多女人结婚后就不化妆了，男人满足了就不奋斗了一样的。

那这个存在感，个人理解为一个人活着的意义，也等同于志向、梦想、自我实现类似的词语。

而以存在感这个角度切入，就容易解释了你是什么样的人，就会有什么样的朋友，或越来越长大，越发现能够交流的朋友越来越少的原因了。

2、对恐惧的新理解

恐惧是对于愤怒的升级。

一个人表现出愤怒的原因在于感觉到自己的边界被侵犯了。

人的边界就是自己存在感的边界。

比如说房子被强拆了，你会愤怒，可如果是被台风刮没了，你就会恐惧。

不同的人，愤怒和恐惧的点是不一样的，那些英雄人物，在常人会恐惧的点，他是表现出愤怒的。比如《勇敢的心》里的华莱士，面对苏格兰统治者的残酷，大部分的人都恐惧了，而他是愤怒，所以他是英雄。

对于英雄的这个定义，很有见地，换句话说看到一个人的恐惧，你就基本上知道他的边界在哪里。

恐惧并不是一无是处的，恐惧是一种身体的保护手段，提醒我们要远离危险。

有时候恐惧还是一种机会。

比如人们会为了解决恐惧，毫不犹豫的花钱：

中国的医疗和教育，是最大的市场，为什么？对生存的恐惧；

为什么医美产品比普通化妆品贵那么多？对青春流逝的恐惧；

听逻辑思维、用得到，为什么？认知焦虑，还是恐惧。

也就是说那些恐惧的其实都是商机。

而巴菲特不是也说过：

别人恐惧的时候我贪婪，别人贪婪的时候我恐惧。

另外，焦虑其实也是一种恐惧。

焦虑是来自于对恐惧的想象，因为是想象，所以无从逃避，所以是一种持续的恐惧，就会内化为焦虑。

恐惧和焦虑是最影响内分泌稳定的情绪，睡眠减少、食欲下降，甚至性欲消失……极度的焦虑和恐惧，真的可能产生“一夜白头”。

在这个迅速变化的世界，每个人或多或少都会有点焦虑吧，如何解决焦虑呢？

既然焦虑是恐惧的一种，那么照梁宁的观点就是“直面恐惧”。

最后

我之前的理解，一个人愿意做事，也就是双手肯劳动，基本上吃是没有问题的。

如果一个人有做产品的能力，那么对于安身立命这种事完全是小case。
