学点经济学

发表于 2017-07-29 | | 本文总阅读量 次

终于看完了薛兆丰的《经济学通识》，直接说结论：颠覆了一些价值观，值得一看。

一个很残酷的现实是，我自己对经济学的理解其实是非常的肤浅的，看不透这个世界为什么是这样，为什么？

比如珠海的房价（因为人在珠海郊区），从14年前的四五千到17年接近两万，可以涨的那么的变态，为什么15年股市明明前段时间还是牛市，过了不到一个月，熊市就来了。

事后诸葛，经常感慨，如果……就…… 但潜意识我也知道，比如就现在，我就是看不懂未来房子是会继续涨，还是一买就跌，或者某只股票明后天会涨还是跌，如果我又预知未来的能力，我早就“all-in”了，就像以前老家那边的人的赌“六合彩”，他们总是觉得自己对“特码”胜券在握，因为看了那么多资料，有的是经过电脑计算，然后顺理成章的倾家荡产了，没有死的都是传奇。

因为现在的记忆已经能够追溯到小时候，在我年少时代，是没有电脑没有手机的，那时候1毛钱是可以买冰棍糖果的年代，而现在1毛钱掉地上估计某人连弯腰捡起来的耐心都没有了。以前的“万元户”抵得上现在百万富翁。同样是100 元能买的东西越来越少了……

知道一点点关于经济的知识，很片面的认为钱薄了，是因为所谓的“通货膨胀”惹的祸，房子涨价是所谓“炒房团”和政府控制土地，都是黑心政府变相收税，而中国的A股是因为背后庄家不按套路出牌。

后来看到人说通过理财也是可以赚钱，因为有些人通过投资，可以让钱生“儿子”，牛逼的如股神，一般的比如身边的“股神”，可经过了15年的骤然崩盘，自己实际能够想到最保险的现在就是存到“支付宝”里面，因为马爸爸财大气粗，应该不会把我等屁民的小本金吞了吧，赚的少总比没赚好。

也就这样了而已。

这些都是很钱有关系，官方高大上会说这些都是和经济有关的，那么很容易想到如果知道一些经济的规律或许就是可以顺着趋势，不能撞上风口上的猪，也能够趋利避害，做好防身也是好的，让赚到手里的钱不会不动都变薄。

很明显，我需要也该学习点经济学的知识，不求成为专家，也是需要了解基本的原理，所以上网去找，看人推荐，找到一本曼昆的《经济学原理》，这个记得大学室友就有一本实体书，就它了。

可是经济学太学术了，好几次翻开曼昆的《经济学原理》，可是就是看不下，后面还找到有插图本的……是真的看不下去啊！

知道得到的专栏，有个叫做薛兆丰是讲经济学的，订阅人数超过十万。我之前都不知道有这号人物，即使他是北大的，好像很厉害的模样……原谅我在自己的小圈子里孤陋寡闻。

得到的专栏，我的印象就是：这些专家其实只是把自己的一本书拆开来讲给人听。我觉得老罗很有智慧，我并没觉得他们讲得不好。只是这个app，即使是音频，用听来解决碎片化，可我尝试听过，还是觉得这非常碎片化，甚至更乱了，因为他天天提醒我要学习这，学习那。

即使是“免费”的，好多没看的，给人感觉还是压力山大，干脆不看了，而这其实也不会对生活产生很大的影响，因为自从静下心读书之后，以前的那种“怕错过”的心态减少了不少。

当然这种学习方法可能只是不适合我而已，我总觉得不成体系的学习，天天碎片“学习”，就像那些天天嚷嚷着要减肥健身的，这都是自我安慰而已，因为这都是在“系统”内变化，起不了太大的变化的，这个问题也好大，这里不展开了。但是，有一点可以确定的是，大众的眼观是雪亮的，这些专栏的作者经过了市场的考验，就像买日用品，我们总是想要买品牌好的，这样才能够确保自己买的确实是质优价廉的好货。

经过这样的思考，我想我应该看一下薛兆丰写过什么书籍，搞来一本，很容易就找到了这本《经济学通识》了，废话不多说，花了一周空闲时间终于啃完。里面的观点有些确实很有启发，有些甚至是颠覆，我被说服了，比如说：

私有物品

高速公路，市内道路，公园，火车这些是属于公共区域，在我印象里这些“理所当然”是属于公共物品，你可以用，我也可以用，完全“正确”。但理所当然的“正确”，并不代表一定正确，经济学的定义的私有物品是这样的：

公用品，指的是一个人用不影响其他人的物品。典型的例子是音乐旋律、故事情节、科学定理等。公用品既可以由政府提供，如公共电视台的节目；也可以由个人提供，如带版权的电影和书籍。

私用品，指“一个人用了别人就不能用”的物品，包括粮食、电力、用水、医疗服务、教育设施、国家公园、交通工具和公路航道等。私用品也是既可以由政府提供，也可以由个人或私营机构提供的。关键是，不论谁提供，也不论提供者是否向使用者收费，私用品的“一个人用了别人就不能用”的属性不变

不论高速公路还是市内道路，都是具有“一个人用了别人就不能用”的性质的私人物品。从经济学看，道路不是公用品（public goods），而是私用品（private goods）。

这样子定义公有和私有才是触及本质的，理解了这个，才会明白春运火车票的问题以及一些公共政策后面的利益博弈。

价值

很多人都有这种想法，比如上班时候，我做的活越多，那么这个项目我的贡献就很大，没有功劳也有苦劳啊，谁干活多谁“价值” 大，因为至少态度在，作为老板看在眼里，做事的态度就是这个人的价值体现，而这又体现在加班上。我一直觉得这好像哪里有点不对，因为之前读书少，确实也想不通怎么回事。

当我看到以下这段：

相信“劳动力价值论”的人以为，商品凝聚的人类劳动越多，商品的价值就越大。他们一厢情愿地认为，在生产中投入多少资金和劳动，产品就具有多少价值，一个国家的国民生产总值就有多少增长。

可一条好端端的路，挖了又补，补了又挖。在计算GDP 的时候，人们会误把所有挖挖补补的成本，都算作它的“价值”，仿佛投入得越多，它就越值钱似的。但实际上，无论怎样计算GDP ，路也还是这么一条路。人们对这条路的“个人估值”，是不会因为铺路时浪费了巨额投资而有丝毫增加的。

某些沙漠开发工程，要投资数以亿计的资金，铺设蜿蜒的水道，延伸到干涸的沙地，目的是在那里种几棵菜。那几棵菜固然凝聚了千万资金，它的“价值”或许有几千万，但它的“个人估值” ，却只是顾客愿意支付的那几块钱！

而这又引申开，深入这个问题：

“价格”什么时候算作与其价值“相符”，什么时候算作“波动”，什么时候又算作“背离”，其实完全没有标准的。“价值”这个概念的缺陷，就在于它是个空中楼阁，既不能证实，也不能证伪。

真正有意义的是”个人估值”，比如我想珠海的房价其实并不值，其实对应的意思是，我个人对某件物品（房价）的“个人估值”，就是我为了换取或保有这件物品（房子）所愿意付出的其他物品比如（金钱）。而市场并不是这样子的，不由得我的主观决定。

回到刚才那个问题，功劳需要赞赏，苦劳应该鼓励，但苦劳怎么也不能掩盖功劳，这种判断不能因为做的事多而本末倒置

只知道数据，可能是相反的情况

很有体会是在《成长，YC创业课》这本书，我看到了这么个例子，

美国的犯罪率连年下降，连续三十年……人们凭直觉给出了很多理由：警察局说是他们冒着生命危险工作才有这样的结果，大学说是它们的教育培养出了高素质的下一代，非盈利组织说是他们的关怀和努力起了巨大的作用……结果，芝加哥大学的 Steven Levitt教授，用详尽的调查统计数据告诉我们，真正“相关度”最高的理由是三十年前，美国通过了允许堕胎法案，最终，那些可能的犯罪分子在还没有出生的时候就已经被干掉……

书里面也举例很多其它的例子，比如：

常洗手的人很干净吗？未必。假如他终日跟脏物打交道，即使常洗手也是脏。常洗手的人很肮脏吗？当然不是。假如他洗手是洁癖驱使，那么他的手比一般人干净。

死亡率高的地方不适合居住吗？未必。加州死亡率居全美之冠，是因为那里气候宜人，最适合养老。死亡率高的地方就适合居住吗？当然不是，在英国工业革命时期的矿区，死亡率很高，以今天的标准看，那不是人住的地方。

不花钱看病的人身体健康吗？未必。很多穷乡僻壤，几乎没有医生和药物，即使眼睛瞎了，牙齿蛀了，骨头折了，也得不到医治。花钱看病的人就体质孱弱吗？未必，发达国家的医疗费用占国民收入的比重越来越大，那里的人活得越来越健康，越来越长寿。

洗手的频率是数据，死亡率高低是数据，医疗费多少是数据，但知道这些数据，并不是表示知道这些数据背后的情况。同一个数据，既可表明一种情况，也可表明与之相反的情况。如果只知数据，不去了解情况，就会得出荒谬的臆测。

看到一个数据引发了一个“顺理成章”的结论，往往并不正确，而这个真的得有条件，也就是说“看情况”

觉悟

经济学其实是通往真实世界的方法论啊，这真的该学。
