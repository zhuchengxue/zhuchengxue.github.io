再谈“第二序改变”

发表于 2018-03-03 | | 本文总阅读量 次

其实这个世界就是不公平的，一个赤裸裸的现实就是，人是有阶层的。

这个世界，这个社会，很多都是由多重复杂的系统构成的。比如人的认知水平。

浅谈认知水平

比如一个非常常见笑话：

三个乞丐冬天在巷子口讨饭。甲对乙说：“我要是皇帝就好了，我就让公差把这条街的剩饭全部都收来归我，不用讨。”乙说：“你就知道讨饭！我要是皇上，我就打个金斧头，每天砍柴去。”丙嗤之以鼻：“你们两个穷鬼！都当了皇上，还要干活吗？让娘娘们天天烤红薯伺候我吃！”

生活在低水平的人很难想象富豪巨贾的生活，现在流行的那句话“贫穷限制了我们的想象力”说的就是这回事。

这种故事笑笑就罢了，真正让身为现代人动容的是同一个信息比如新闻的不同人的理解不同，进而导致的行动完全不同：

比如在知乎看到了一个关于红黄蓝的讨论：

有人至今都认为有军队人员参与其中；

有人一开始就知道的确涉嫌有虐童，但不涉及军队，系家长为了求关注度故意造谣给军队；

有人从中看到了民营幼儿园的安全漏洞，进军幼儿园安保、摄像头行业；

有人提议立法保护幼儿在幼儿园的人身安全；

有人想到了如何提升幼儿园从业者素质；

有人直接到美股市场做空红黄蓝股票，赚到一大票。

不同的阶层，不同的思维方式，不同的看待问题的角度、眼光，都极大地影响着我们对于信息的筛选、处理能力，也就决定了我们大多数人的认知能力。

但这并不意味着层次越高的人，看问题越全面，比如一个我们熟知的笑话：

有一年发生饥荒，百姓没有粮食吃，只有挖草根，食观音土，许多百姓因此活活饿死。消息被迅速报到了皇宫中，晋惠帝坐在高高的皇座上听完了大臣的奏报后，大为不解。“善良”的晋惠帝很想为他的子民做点事情，经过冥思苦想后终于悟出了一个“解决方案”曰：“百姓无粟米充饥，何不食肉糜？”(百姓肚子饿没米饭吃，为什么不去吃肉粥呢？)

但即使这个傻逼皇帝居然还是个不错的人。有人替他伸冤，说他很善良，对人也和气。
他并没有逃避做皇帝的责任——喂饱黎民百姓。他还是有颗想要解决问题的心，只是不知道解决问题的方法。
例如：嵇绍的故事。

晋惠帝被皇太弟追杀，一路狂奔逃命。自己都身中三箭。史书记载：百僚奔散，唯侍中嵇绍扶帝。当追兵追上了，要绑架他回去。晋惠帝，这个历史上的白痴皇帝做了一件很让我震惊的事。他求追兵饶了嵇绍一命。

在自己生死未卜的情况下，他想到的是救下曾经帮助过自己的恩人。但追兵还是当着他的面杀了嵇绍，血溅了晋惠帝一身。

事后，追兵要给他换衣服，他吩咐别人这样一句话。他说，这件衣服上有嵇绍的血，不要洗去。他要留着这件衣服。即便嵇绍死去，晋惠帝也永远不要忘记自己的恩人。

通读中国历史，这样记得别人情义的，别说是皇帝，就是一个普通人，也是少有的。

所以即使是一个贻笑千年的人物，都可能还有我们不知道的另一面。

多重系统

在上一篇的《破局的智慧》，我们知道最终的改变在于实现“第二序改变”，不然就像在睡梦中不断的逃跑、跳跃，还是难逃噩梦，最有效的是从梦中醒过来，才是最有效的改变。

比如古典就把勤奋就分为三个层次：

第一层：很努力

第二层：方法论勤奋

第三层：更少目标，战略勤奋

每一层都是其上一层的第二序改变。

这个观点对我的启发非常大，虽然以前也知道目标很重要，男儿要有志向，可是很多时候都不当一回事，理解层级的不同很大可能决定了现实中的阶层。

又比如写作这件事，我总是想着只要一直写，想到什么就写应该就可以达到一个非常不错的结果，也就是那种“但行好事，莫问前程”的心态。

但如果把写作这件事，划分层级结构：

细节——句子——段落——框架——主题

最好的写作方式是自上而下的。正像古典所说的，其实我们很多人在写作不顺的时候，常常不小心掉入低水平勤奋的陷阱。比如开始构思一篇文章的框架，这其实是最重要的部分，但实在费脑子，

于是我们开始查资料（最底层），来回修改其实没有什么区别的文字（句子）或排版，往往这就能搞半天。

有时候更糟，我们反复“调整状态”，收拾东西、擦桌子、吃东西、看一集美剧，充实地干了一天，结果当然一点儿都没有改变。

这就解决了之前的那个问题“不识庐山真面目，只缘身在此山中”，跳出来就好了，到更高的一个层次看问题。

需要解决的问题，就像是被锁上的锁头，而解决方案就像钥匙，钥匙一定不在锁孔里插着，一定在别的什么地方。所以当我们尝试解决任何问题时，只盯着问题看和想，盯着问题找解决方案，通常只能以无奈告终。

时刻提醒自己：一旦你自己遇到问题时，发现自己正在“只盯着问题本身思考当我们一直想着要开锁的钥匙的时候，我不能这样，我应该把注意力从问题本身移开，因为解决方案肯定在其它地方。

最后

正如《跃迁》这本书里面所说的：

一定要相信：你今天遇到的问题，早就有人经历过，并且找到了更好的方式，你说要做的，只是学习。
