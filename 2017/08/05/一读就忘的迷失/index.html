一读就忘的迷失

发表于 2017-08-05 | | 本文总阅读量 次

迷失

肯定不是我一个人有这种感觉，如果是一些非虚构（no-fiction)的书籍，经常看了后面，忘了前面，一本书翻完了，都不知道这本书讲的是什么。

好长一段时间我都陷入了这个迷失，这样看书把自己脑袋搞成一团乱麻，真的是死读书，活受罪啊！我仔细想过这个问题，真的，一度怀疑是不是自己的脑袋不行了……

一读就忘的书特点

记得以前看小说那么大本砖头书，两三天啃完，还意犹未尽，而看不进去的书也就两三百页可就是很难内化到自己的脑袋里，想想也真的尽力了，这些书包括：

  • 外文原版（不限于书、名著、杂志、网页等）
  • 翻译过来的畅销书
  • 专业的书籍

或许有一些原因：

  1. 外文书籍非母语，背景知识，单词，语法等都有差距，看一页花一个小时，意志力吃不消；
  2. 翻译过来的词不达意，或自己理解能力太差；
  3. 专业知识非常的枯燥，难啃。

但我们实实在在的知道啃这些书是可以让我们成长，提高我们的能力，所谓的充电，也就是通过充这些书到脑袋里才是真正的完成自我救赎啊！

一定有办法的！念念不忘，终有回想。我觉得找到了一些技巧来克服这个问题了。

绝对不能按照读小说那一套方法来读这些书了

想想我们读这些书的目的，很多时候是为了掌握技能，就像我们学习使用一种新的仪器设备，我们一开始是看说明书的，有条件的看视频或听老师讲解。而当我们阅读说明书的时候，我们很多时候是不会一个字一个字的从头到尾的看下来的，大部分我们会这样：

  1. 先看这个说明书主要的结构—目录，看有几个部分构成；
  2. 边看边用
  3. 多用几次就熟悉了。
  4. 我们把说明书当做地图，而不是当做收藏。

当然非虚构书籍并不是简单的一张纸的说明书，但大体结构很多类似，这体现在技能类的书籍里面。那我们可以做的先整体上熟悉这本书到底讲了啥：

  1. 先看豆瓣或亚马逊书评，很多的“剧透”，评价，这个不是小说电视剧不怕剧透，我们关心的是我们花了时间下去到底能不能有收获。
  2. 谷歌搜索作者的介绍，以及这本书的相关资料，类似的书籍也可参考；
  3. 打开这本书，先从整体上纵览全书，看目录，大标题小标题
  4. 以最快的速度先把所有内容串起来，不看例子，只看结论，这条线的连接非常重要。因为我们真的会很容易放弃的。
  5. 时刻做标记，觉得有所悟，或者自认为很重要的先摘抄下来，这个就像走一条陌生的路径，我们会做好一些路标免得迷路一样。当我们合上书籍的时候，脑袋一片空白的时候，我们觉得自己好像什么都不记得了，其实不是的，记忆有缓存在工作记忆，没有到长期记忆，这时候只是提取很困难，但别忘了我们有摘抄，它会像路标一样指引着我们回忆起大部分我们看的内容。

不可能只看一遍的

经过上面的一番快速概览，我们基本上可以确定这本书是什么质量的。既然是好书，而且你看了有一读就忘的特点，那么这本书就不可能只读一遍了，我觉得至少得再读一遍，最好两遍，在第二遍的时候做好相应的记录，我已经不相信自己的记忆力很久了。

刻意阅读

一个常识：高手都是刻意练习出来的， 看读透这本书，必须刻意阅读了，该怎么做呢？

现在的我是这样搞的：

  1. 知道哪些是重点目录，第二遍先看，仔细认真，做笔记，查资料，只字不差的阅读；
  2. 遇到不懂的名词，先谷歌图片看是什么鬼，再wikipedia 了解功能；
  3. 动词尽量查与之关联的介系词，套到句子里面很多就可以理解整个的意思了。
  4. 时刻提醒这本书讲的是什么，作者的逻辑是怎么验证的，这是真的吗，这是对的吗？
  5. 读书看完也就只是看完了，看完之后，至少要问自己，我从中可以做出什么改变（新技能，新认知，新世界）？
  6. 写下自己的感想，形成博文。

一点经验

  1. 要看书就要远离干扰，手机静音，电脑断网；（我觉得很有必要，我的自制力真不行）
  2. 人的情绪会很消化神经冲动能量，尽量保持心平气和；
  3. 天下武功无坚不摧，唯快不破，尽快看完概览一遍；
  4. 着重刻意阅读；
  5. 注意锻炼身体，久坐伤身；
  6. 原谅自己，其实没看也不会对自己目前的生活产生很大的影响。
  7. 纸质书和电子书都有各自的好处，眼睛看电脑累了，可以看纸质书。
  8. 看书只是输入，一定要有输出……不然只是工作记忆，迟早要幻灭。

要知道，一读就忘的书不是一般人可以看的，那么反推能看这些书的人绝对不是一般人，共勉！
