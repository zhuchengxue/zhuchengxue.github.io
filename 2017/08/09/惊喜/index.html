惊喜

发表于 2017-08-09 | | 本文总阅读量 次

读书可以制造惊喜

读书的惊喜在于：

默默看书的时候，不经意的时候突然想通了一件想了好久的问题，欣喜若狂，这很辛福，我确定。

这也许就是那些人所说的，念念不忘，必有回想吧，现在的我相信了。

不能聆听这个时代智者的言传身教，但是通过阅读这个时代的智者所写的书籍，可以更好的把握自己的人生。

别人比我聪明，比我努力，比我起点高，早就把一些问题想得清清楚楚，明明白白，都可以形成标准操作规程了，虽然伸手党真的不好，但通过自己的思考与努力，我相信那是真的正确的。

这就像看电影或电视剧剧透的时候，你会发现，剧情的发展和剧透的情节实在吻合哈，你知道会这样，你也很认可会这样，你在等着他真的就会变成这样，你心里会心一笑。

这真的是惊喜，最最让我感到惊喜的是复利这个概念。

复利可以成就希望

在最基础的理财书籍里面，亦或是追溯到我们上小学的时候，就学过复利的概念，复利的概念非常简单，其实就是以前的利滚利，但没有亲身经历的话，对于复利这个概念是很难有深刻的领悟的，复利的关键有两点：

  1. 成长率
  2. 周期

尽量提高成长率，缩短周期，使得n的数值越大越好。我讲的很不清楚，这个得让高手来讲，推荐李笑来《通往财富自由之路》里面介绍，论把别人说服的能力，我真的服他。

复利很大原因让我们迷惑的是它的一个属性—长期，可是长期并不是绝对的，对于高成长率的公司或个人，他们的“长期”可以缩短，这样推导下来，会发现，能力强的人最终会更“长寿”，因为完成相同的事情，能力强的人，可以更快更好的完成，而且可以实现更多的时间自由的做其他人一辈子都来不及做的事情，相比之下，他们确实是更加的长寿。

我其实可以更早达到的

之前的一些定见，或者已经知道的知识，会让人心浮气躁，刚看前面两段，就会觉得，自己可以猜到作者要讲什么了，然后就囫囵吞枣的看完，因为是扫读，也真的看到了“自己认为”的内容了，然后心安理得的合上书籍，再去看其他的了，可是当有机会重新再翻看的时候，里面的某一部分，非常的重要，非常的精彩，几乎颠覆了之前的认知了，好后悔为什么之前没有注意看呢，现在已经过去了一年了，相当于损失了一年的行动，而这又会损失多少的收益呢，相信每一个人心里有一杆称。

可是年轻人就是急躁啊，总想要赶紧多充电，多学习，多看书，努力的赚钱，然后把一些基本的生活物品都满足，停不下的脚步，看不完的书籍，上不完的班……总是有很多的理由的。

放轻松，有些事情没办法的，好书不怕翻，真金不怕火炼。它也真的值得翻，可以把之前错误的认知，或者不懂的全部扫清障碍。

在智者面前，内心要极为虔诚，完全接受他们智慧洗涤，沐浴认知的成长。因为这可以达到野蛮成长，而自己内心的那点微弱反对的声音，如果真的认真听下来的话，会马上再也不会想要说出来的，因为这太丢人了。

坦诚的重新定位自己，这不是网络上的不忘初心，这就是认认真真，真真切切的剥开自己的问题，分析解决之后，定出行动的计划，其它真的就是按部就班了。

每一个作者在他的一生中可能写过非常多的书籍，但是质量可能参差不齐，如果对于逻辑结构非常强的，比如小说构架太过宏大的，不一定越老越值钱，相反如果是同一个主题，经过更长时间的思考的，这个作者的新的书也必然能够跨越一个新的境界。 如果真的想要节省自己的时间，可以考虑只是看最新版的，但相信我，你会顺藤摸瓜想要看作者其他书的，即使后面发现最开始的他其实也是有点“笨拙”。

对即使是很厉害的人，可以称谓这个时代的智者，也是从无知到如今很厉害的状态，这需要过程。

所以，少一些事后诸葛亮，多一些反思与反省，切合世界的做一些力所能及的事情，这就是本分做人。

今天很开心！
