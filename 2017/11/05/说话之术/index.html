说话之术

发表于 2017-11-05 | | 本文总阅读量 次

生活中只要需要和人打交道，不可避免就要和人说话，一个普通人，小到去商场买个菜砍个价，大到公开场合演讲，都要扯几句话。会不会说话，懂不懂说话，关系到你会不会被宰，会不会升职加薪，会不会有人缘……最终，你怎么说话，在别人的印象里，你就是一个什么样的人。

不会说话

不会说话会带来什么后果呢？《好好说话》这本书里面有一段精辟的论述：

人生中有没有经历过一些灵异事件？

莫名其妙，有些人开始疏远你；莫名其妙， 有些邀约人家就是死活不同意； 莫名其妙， 你会碰到一些或明或暗的软钉子。 最可怕的是， 你甚至根本不会意识到这一切正在发生， 而只是隐隐觉得不顺、 隐隐觉得哪里不对。

说话之伤，都是暗伤， 自带缓释效果， 若无人点醒， 至死不知。能意识到你问题所在的，通常都不会告诉你。心大点的， 察人之过笑而不言，但是知道你这人不靠谱，以后有重要的事不能指望你，就会疏远你；心稍微小点的跟你计较，但是客客气气找个别的由头拒了你；等而下之的人，甚至会一边给你穿小鞋一边装大度，让你死都没死个明白。

日常生活中，有很多看似微小的冲突，由于沟通不良，在日积月累后都又可能成为人与人之间的不定时炸弹。说话之伤，都是暗伤，天性善良，容易木讷；天性聪明，容易尖锐；天性口无遮拦，容易伤人不自知；天性柔和的人，容易絮叨惹人烦。朋友圈无意中的炫耀，吹水聊天中无心揭人所短，人之本性，得意时容易忘形，失落时随处抱怨……

有多少次想表示关心，在别人听起来却是指手画脚；想表示安慰，在别人听起来反而是漠不关心；想表示委婉礼貌，别人听起来却是暗藏心机。得罪人这事未必非的起了正面冲突、口出恶言，很多时候，我们即便心里并没有什么恶意，但表达和思路上出了问题，还是会让人不舒坦，而对方还不好跟你一一摊开来计较，以至他心里疙疙瘩瘩，变得更加不舒坦，只好对你敬而远之。

我自己是一个内向的人，自我定位就是一个超级不会说话的人，说好听点就是心机少，说难听点就是嘴巴笨。很多时候想到什么就说什么，内心深处又一直觉得只要自己以诚待人，出于公平的心态，别人也会这样待我，自我感觉还不错，现在想想可能别人把自己当傻瓜耍而不自知。后面阅历多一点，隐隐觉得有哪里不太对劲，感觉是祸从口出，就慢慢相信沉默是金这种说法。因为不屑于见人说人话，见鬼说鬼话，那种阿谀奉承姿态自己又根本做不到，可能早就落下了如前面所述的暗伤。对于不会说话这件事我是“经验丰富”，在看这书的过程中，时不时的会想起自己之前说过的蠢话进而面红耳赤不得自己。

什么是会说话

用第二届《奇葩说》 冠军邱晨的话来说：

兼具“耐思”（ 进行深入的思考和分析的智慧） 、 “耐撕”（处理观点交锋和利益冲突的能力）、 “nice”（与人为善追求达成共识）， 才是这个时代所要求的“好好说话”。

谦和而又自信、善辩又能守拙、光彩夺目又机锋内敛、 能锱铢必较也能把酒言欢。这个道行有点高，里面涉及了交流沟通，谈判，演讲，辩论之要素。

比如说，说服别人这件事。说服的关键，就是要尊重并且利用对方的自主性，使其从自身角度出发，心悦诚服地接受我们所希望其接受的观点。我们要卸下对方的心理防御机制，诉诸对方自身的心理需求，让“我说服了你”变成“你说服了自己”，让“我的观念打败了你的观念”变成“你的观念升级了”。比如现在所有“知识付费”，在赚了你的钱的同时，还会让你觉得自己赚了，因为自己的观念升级了，当然这里可能也确实有实打实的更先进的观念，双方达到共赢。

不同的场合说漂亮的话，这是一种功力，这并不是天生的，需要后天的练习。举3个例子：

1. 会说话有时候是换一种方式表达

比如我们想知道别人有没有小孩，有的人会这样问：

“你有小孩吗？”，“打算什么时候生”，“小孩多大了”

这种单方面的索要信息的问法，会给人感觉像是在接受拷问，换一种方式：

“你喜欢小孩吗？”，“这年头，当父母真是越来越不容易！”

在比如想问别人有没有买房或有没有女朋友之类的问题时候，稍微调整下改成

“你觉得现在这个时候投资房子可行吗？”，“现在女生真麻烦，还不如一个人呢！”

换种方式，把态度问题抛出去，对方自然会聊到我们想要的信息了。

2. 扯到钱很不好说话

遇到亲朋好友借钱，有时候是一件挺尴尬的事情。对于钱，我们知道它的一个重要属性天生就会产生利息的，而和它相关的所有事物往来都逃脱不了权衡利益。

一般借钱理由不外乎两种：

一是救急， 比如说家人生病要借钱救命， 这个时候我们的态度往往是能帮就帮。 如果对方借钱的数额实在太大，借不了那么多， 或是压根儿不想陷入无止境的催钱与拖债中， 也可以将你认为合适的金额用包个红包的方式送给对方，而原本是想借钱的亲友也不太可能挑你的理。

二是免息， 比如说对方是因为买房、 买车等一切购买行为而借钱， 目的是为了省去银行利息， 把我们当免息并且无限期还款的银行在用。 面对这种理由， 一旦开始借钱， 基本上以后双方的关系就是催款关系；不借， 反而可能还有一线生机。

当对方说出诸如买房之类的借钱理由时，不要马上说“买房啊，不借”这样让人尴尬的话， 而是要进一步以关切的态度来定对方缺钱的原因， 比如：“买房首付不够啊，理解理解， 现在房子的确太贵了。 ”千万不要傻乎乎地直接问对方，凭什么买超过自己支付能力的房子，这样显得很不客气。 只要我们开始感叹房子太贵， 跟对方做出同仇敌忾的样子， 对方一定会顺势进一步解释不是自己非要超前消费，而是丈母娘或者老婆逼的，这就间接帮助我们确定了对方借钱的根本原因及压力所在，同时也为提出拒绝理由打下了基础。而探听清楚原因后，接下来， 我们就可以根据对方的原因给出一个不借的相似理由，

比如：“我借钱是因为最近想筹划买房，你看能不能帮我周转周转。 ”

“哎呀， 怎么这么巧， 我恰好最近也在计划买车， 手头的资金压力也很大， 想帮你也是有心无力啊。 ”

此时， 大家都是为了自己的美好生活在努力攒钱， 对方脸皮再厚也不会说“你的美好生活先等等， 我的美好生活比较急”吧？

当然借钱给人从来都不是义务，没什么不好意思的。你肯学习，好好说话，把拒绝的话说得好听些，已经是在额外付出了。

3. 想要抱怨，先打“预防针”

由于沟通不良而导致的冲突、误会，比如“婆媳关系”如何改善呢。人总是以自我为中心，旁人劝说的时候经常“要多体谅长辈/要原谅后辈不懂事”，其实是希望双方能够换位思考，可当人在情绪中的时候，这是无效的。要改善这种关系， 最重要的不应是后知后觉地弥补， 而是要针对冲突产生的机制进行预防。 要弄清楚， 那些看似微小的冲突到底是怎么产生的， 又是因为什么原因而没能现场解决， 从而积累成了人与人之间关系的裂痕。

其实最主要的原因在于我们意识不到一件事情存在“两个版本”。

什么叫“两个版本”？就是我们每个人在面对同一件事的时候，内心都有一个专属于自己角度的版本。 在我们自己所编写的版本中， 所有的行为都是有原因、 有苦衷的； 而所有外在的、 来自他人的反弹与质疑， 对我们来说都是缺乏体谅的、 胡说八道的， 甚至是挑衅的，虽然我们在沟通时都会强调“我对事不对人”，但在每个人的心中，对于事实都有一套属于自己的版本。

所以， 我们就要了解， 所有的沟通事实上都是在“对人不对事”。 在沟通中， 我们对于事实的每一个陈述， 都是在以某种形式攻击那个拥有不同版本的人。囿于情绪和认知的不完全，我们往往意识不到“两个版本”的存在，而都误以为是对方不讲道理。 这时候， 冲突就不可避免地存留下来甚至爆发出来。

既然认识到了“两个版本”的问题，我们就要努力去拆除这个陷阱，而最简单的方式就是。打一剂“感受的预防针”。

比如， 当你想要宣泄情绪、 表达不满之前， 试着做一些这样的铺垫：

“抱歉， 这或许未必是事实， 但我难免会有这种感觉。 ”

“抱歉， 这或许未必是事实， 但我难免会觉得你这样做是对我的一种不尊重。 ”

之所以用“感受的预防针”来为之命名， 就像为了抵抗疾病， 小孩需要打预防针一样， 面对某些敏感的沟通， 我们也会需要预防针。 而这句话的妙处在于， 它预先向对方说明“我接下来要讲的仅仅只是我的感受而已”， 也就等于是在暗示说“我讲的有可能不是客观事实”，从而大大降低了会给对方带来的被冒犯和被攻击的感觉。

最后

说话要分场合，不同的情景下，不同的说话对象，该怎么说话都是很不一样的。《好好说话》这本书里面有很多的应用场景，有兴趣的可以去了解下。我自身一直信奉真诚是最好的话术，但是好的话术可以更好的实现我们想要的结果，还不伤人，这值得学习。

最后，上次回家我的父亲说过这么一句话：

“没用的话不要说，因为没用；没用的事情不要做，也是因为没用”

有用没用个中标尺，只能自己衡量。好好说话是一个技术活，于我来说，路漫漫其修远兮……能说好一点是一点吧。
