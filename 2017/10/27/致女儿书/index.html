致女儿书

发表于 2017-10-27 | | 本文总阅读量 次

亲爱的伊诺小宝贝：

今天你满月了，时间过得真快，爸爸为了当一个称职的父亲，最近在看《傅雷家书》，恶补为人父应做的事情。上世纪50年代的家书，傅雷拳拳父爱真真切切，现在读来还着实让人感动，初为人父，因为不在你的身边陪你，也想写封“家书”与你。

你的降生，补充了爸爸的回忆，我能够记得的最早的事情可以追溯到4~6岁之间，在那之前的记忆（从娘胎到这个世界）却是一点也不知道了，最开始的最开始，即使我很想回忆都不可得。很自然的联想，原来，每一个人以前都是这样的。这是一个从无到有的过程，而且这是有灵性的灵魂诞生的过程。

但即使了解所有的过程，一个和我有息息相关的生命的诞生还是给了我很大的震撼。我想把这个体验记录下来。

在你降生之前，作为即将成为父亲的我来说，并不知道要做什么，并不知道是什么感觉，压力还是喜悦？我也说不清楚，生孩子大部分都是女人的事情，也就是你妈妈的辛劳。我能做的和我真做的事情非常少，在使妈妈孕期稳定这件事上，我其实做得并不好。

很多人问我，快要做爸爸了，第一次感觉是什么样的？

老实说，应该是压力，责任占据大部分。我总感觉自己还没有准备好，特别是迎接你的到来，因为让我很忧伤的一件事是，对于赚钱这件事，我做得并不是很好。相反还很糟糕，不论是投资还是工作，都没有一个非常明确的方向。

似乎你也知道爸爸的顾虑，过了预产期还是不愿意出来，我们隔三差五就去卫生院做检查，每天行走10000步以上，“三九都”是生我养我的地方，可是有很多的地方我其实并没有去过，这次因为你这小淘气，我居然走了其中的大部分，一次美好的意外。

又过了差不多一周之后，因为一些顾虑，我们决定“催”你出来，当然不是刨妇产。很庆幸，并没有什么差错，悬着一颗心终于放下了。

第一眼见到你的时候，刚从产房推出来的时候，你眼睛都没睁开，看到你粉红细嫩的皮肤，一只胡乱摆动的小手，一声啼哭，那个声音非常清脆响亮，恩，我真的当爸爸了……内心百感交集。

你居然会笑了，我都后悔没把你的第一次笑容记录下来，但我可以很容易的想到你那时候的样子，内心的感觉很是温暖，嘴角不经意的露出微笑。

对于时间的感知，让我很清晰的可以知道，很快，你就会长大了，就像家里的姐姐，哥哥们，你会开始走路，去上学，从丫丫学语开始，幼儿园，小学，初中，高中，大学……你会交男朋友，会结婚，会经历身而为人所有的痛苦与快乐。当然你并不会走我走过的路，未来的社会会非常的不一样，爸爸从小到大和你爷爷从小到大所经历的完全是不一样的世界，这个世界变化太快了，感觉每隔不到十年，世界就翻一翻。

我很容易的想到，你这么小人儿，就是一块璞玉，细心雕琢，你完全可以成为任何你想成为的人，非常大的可能，你会长成我从来都没有想到的样子。

我并不是一个会做计划的人，一个可行性的计划，明显是有用的，可是我不擅长，这是一个缺陷，我要改。不然未来的路很大概率会让我措手不及。在你还这么小的时候我并不在你的身边，幸亏这个时代已经有微信等工具，让我和你妈妈可以时不时的进行视频，我也可以看到你的哭和笑。你让我勇敢，做我以前一直要犹豫的事情，比如去做自己想要做的事情。

回家陪产那段时间，你爷爷说过一句话很是打动我：

没用的话不要说，因为没用，没用的事情不要做，也是因为没用~

所以对于爱你和你妈妈的话，我也真不会说，但就像傅雷说的

只有事实才能证明你的心意，只有行动才能表明你的心迹。

的我知道该怎么做了，我想把所有我知道有用的话和有用的事情都告诉你，这也是作为父亲应该做的事。

回顾我已度过的日子，辛苦，劳累，喜悦，忧愁，人生这条路上的磕磕碰碰，弯路，挫折，有太多的彷徨、茫然、无助……我总不想你也一一经历，即使经历的时候，有个打你三十岁的人可以为你指导。

人生匆匆，我很怕忘了，所以我也要记录下我们第一次见面的情形，当我离开这个人世，你就是那个美好的存在。

但你到底会长成啥样子，我也是很期待的。

真正的改变
