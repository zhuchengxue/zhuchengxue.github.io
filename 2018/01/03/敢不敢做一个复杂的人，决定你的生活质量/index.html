敢不敢做一个复杂的人，决定你的生活质量

发表于 2018-01-03 | | 本文总阅读量 次

简单到复杂能够得到品味的进阶

2017年最后几天，朋友圈流行晒十八岁，连习大大在新年讲话的背景中也出现了他十八岁的照片。

这种怀旧现象的产生，马尔克斯有个精辟的解释：

我们这一代人年轻的时候对生活都太贪婪，以致身体和灵魂都忘记了对未来的期盼，直到现实告诉我们未来，和我们曾经的梦想不一样，便又开始怀旧。

我自己以前也经常会想起过去，然后陷入某种淡淡的忧伤……

但现在的我会有意识的去注意某些细节的变化。

比如，男人的帅，不同时候的感觉完全不一样：

  • 小时候看《神雕侠侣》，感觉古天乐很帅，过儿那坏坏的微笑，白净帅气的脸庞，恍如昨日；
  • 接着看了《古惑仔》，觉得郑伊健很帅，纹身，伤疤，46分的发型，兄弟义气，后面跟着一堆细佬，铜锣湾扛把子，平生不识陈浩然，便是英雄也枉然；
  • 上中学的时候，我感觉张国荣真帅啊，当爱已成往事，风华已不再，他真可以说是美艳；
  • 后来，上大学的时候，毫无疑问，我认为最帅的人非周星驰莫属，一个给大家带来无数欢笑的男人，真想看他继续演戏，无可争议的喜剧之王；
  • 第一次看《黄金三镖客》，我就被Clint Eastwood倾倒了，美国人民评价，说他是帅了一辈子的男人，恩纯粹的帅。
  • 直到看到《教父》，艾尔.帕西诺，帅得掉渣，绝对冷酷，面无表情，波澜不惊，高深莫测，我能想到的是男人中的男人。

不同时期对于帅的理解完全不一样，从简单的感官上的帅，到从里到外散发出来的作为男人的品味，这两种帅完全不是一个档次的。

有品位的人追求复杂

我有一个老同桌，我非常欣赏他的生活态度，他是属于那种叫做“认识到生活的真相，依然能够热爱生活”的人。

今年的一次小聚，我们谈到一些习惯爱好，他是一位象棋爱好者，经常找朋友切磋棋艺，并对此乐在其中。

我提出了我一个顾虑：下棋是智力棋艺的比拼，总会有输赢，而输的一方碍于面子上的问题，总会导致不同程度的不开心。

以前我也下棋，正因如此，常常导致了大家情绪上的不开心，所以现在大部分玩一些纯粹靠运气或运气成分大的娱乐项目（例如摇塞子），大家相安无事。

他有个不同的观点，那些纯粹靠运气的游戏，很容易让人产生浮躁的心态，因为很快就可以看到结果，比如比大小，摇一下就能够得到结果。赢了有些许高兴，输了不但不开心，而且最直接的结果是让自己变得心浮气躁。

而下棋就不一样，一盘棋局，如何布局，试探，攻防，棋子间的博弈是智慧的碰撞，而偶得一妙招，多巴胺分泌可以持续好久，在这个过程中得到的快乐是与纯靠运气得到的快乐是完全不一样的。

这样看来，其实人群中有一部分人是刻意去追求复杂。

为什么有些书我就是看不下去

很早以前，我就想为什么有些书我就是看不下去，那些经过时间洗涤的作品，大家都公认的好书，可是为什么我就不觉得呢？

看书是很私人的活动，当然你可能会说，才不要管他人评价，我觉得好的就是好的，我觉得不好看的，别人吹上天了也不可能把我说服。

但人生如果真要说有什么东西很珍贵，那就是体验，而本不该错过而错过，这终究是挺遗憾的。

后来我真的硬着头皮把一些大部头的名著例如《红楼梦》，《百年孤独》啃完，发现这些书基本都由复杂的结构精密交织而成，有些书直接是构造了一个宏大的世界，里面芸芸众生，人间百态依次上演……

当真的看懂了，也就自然就明白个中妙处。原来很多时候，我们理解不了复杂，进而看不到复杂的美。

说到本质上，原来就是因为自己的积累沉淀完全不够理解复杂的事物，然后自以为是的认为是这书太难看了。

或许正如杨绛先生所说的：

年轻的时候以为不读书不足以了解人生，直到后来才发现如果不了解人生，是读不懂书的。读书的意义大概是用生活所感去读书，用读书所得去生活吧。

有些书是需要生活所感才能够读懂的，对于那些一时半会儿读不懂的书，不要心急，先缓缓，或许哪天生活就指引你回到这本书，像遇到老朋友一样。

生活本就是复杂

有些人可能会说，我就是喜欢生活能够简单些，想那么多，那么复杂多累啊！

简单的人单纯，这个世界对他们来说非黑即白，什么事都要分对错，他们不知道这个世界其实是彩色的。

而单纯后面总是跟着无知。

有一些阅历之后，我发现：

要把一件事要做好，做到极致，即使再简单的事情也会变成非常复杂的。

这是生活的真相，生活就是复杂的，因为你看：

同样是杂志：《故事会》是简单的，《读书》是复杂的；

同样是小说：《斗罗大陆》是简单的，《红楼梦》是复杂的；

同样是教材：《初中数学》是简单的，《高等数学》是复杂的；

同样是娱乐：刷微信刷头条是简单的，但下象棋或围棋是复杂的。

选择简单，就选择普通，然后就很自然没有然后了。

有人觉得城市套路深，我要回农村，可是他们却忘了农村路也滑，人心更复杂。

敢不敢做一个复杂的人，决定你的生活质量。
