浅谈“融会贯通”

发表于 2017-06-26 | | 本文总阅读量 次

经常看武侠小说，对“任督二脉”这个词绝对熟悉，这一般是主人公要突破自身极限，武功大成最后需要突破的阶段。打通之后，就可登峰造极，实现飞花拈叶皆可伤人的境界。

后面慢慢长大后才发现，现代人并没有练就一身钢筋铁骨，居然也能上天入地，日行千里，如果可能穿越回去，一把手枪，就可以把古代绝代高手射成筛子。

这是因为人类会发明并使用工具，现代社会，类比江湖，也是存在真正的绝顶高手，它们比的是脑袋的“进化”。

可以想见，谁能在自己的领域里想得快，准，周全，如果别开生面，想前人所未想，做人所不能做之事，是为现代世界的绝顶高手。

这些人厉害在于他们脑袋里的那点东西非常重要，这个东西，有很多的其它叫法，比如“思想”、“主义”、”灵感“、“创造力”、“智慧”“眼光”“见识”“思维模式”“心理表征”等等。这些名词的定义不同，但指向其实是一样的。

很多人内心深处都想成为一个“很厉害”的人，身披金甲，脚踏祥云，叱诧风云。

如何让我所学的东西达到一种融会贯通，最后完成由量变到质变的升华？

举个例子，现在的好多人生导师级别的人都会认同学习英语的重要性，而工作这么多年的经验也告诉我们英语好的机会也是非常多。可是我们为什么学英语？不说考试类的目的明确那种需求，一般人到工作时候还会坚持学英语，不外乎有以下几个理由：

  1. 有的是为了能说一口流利的美音或英音，

  2. 有的是为了能看懂原版外文书籍小说，

  3. 有人想看美剧或电影不需要字幕，

  4. 有人要做行业的顶尖，大部分行业最尖端的技术是英文

  5. 有人要学编程……

好好思考这个问题后，这些需求其实都有一个共同点，就是上面的讨论的“东西”，其中：

  1. 是为了把脑袋里的“东西”用英文来表达出来；

  2. 是为了把原版书籍的蕴含的“东西”理解；

  3. 是为了把电影或美剧这些影音要传达”东西“理解；

  4. 是为了把行业内的顶尖科技这个”东西“消化；

  5. 是为了能用计算机语和电脑交流。

那这个”东西“到底是个什么东西？
个人浅见，归纳了一些特征，比如：

  • 这是一个闭环的体系，没有体系的知识只是散落的珠子，如一盘散沙；这个体系须由始到终，从表到里，从前到后，从左到右，必然丝丝相扣，经得起推敲；

  • 这个”东西“应该是已经存在的，需要的只是人们通过自己的”修炼“自己达到某个境界，比如读书到达多少个量级就会发现好多书讲的内容其实就一样嘛，但是一例多用，举一反三总能够给人一种新奇感，这或许需要个人悟性；

  • 真正掌握这个”东西“高手，基本出手无形，举重若轻；

  • 这个”东西“绝对不是一个记忆机器，当然好的记忆力，对于得到这个”东西“有助力。

  • 顶尖的”东西“决定一个人的段位的高低，扫地神僧扫数十年可能功用比不上一个扫地机器人，不说虚的，发明电脑的人绝对是人类智慧顶峰的存在；

  • 我们苦苦寻觅，不断寻找，好多先人著书立说，希望能缩短这个获得的路径，但很多时候会因为一些生活名利所阻碍。

  • 有的很可能只是套路。

欲得此”东西“，必须上下而求索……也不一定可得。最终的overcome就是所谓的“融会贯通”。

前人总结的路径如下：

  1. 见多识广：包括读书、行万里路及与人交流，这个需要多，认知必然广博；

  2. 深入思考：不断的挖掘，穷尽所有的思考路径，或许能产生质变。

结果肯定因人而异，有人记忆好，有人资源多，有人脑洞大，有人天赋异禀，最终在这个维度也显示出了人间百态，甚至是物种间的区别。

这只是普通人的一点思考，绝对片面，仅供参考。
