折腾不舒服斯基

发表于 2018-03-14 | | 本文总阅读量 次

当一个东西很复杂的时候，那么学习它就需要很大的时间成本，可是如果真能够学会它的话，就会大大的提高我们的效率，会成为提高生活工作的大利器。

以前我不知道，复杂的东西还很容易让我们上瘾，因为只要你深入进去的话，就会发现折腾的乐趣，比如下棋、上网、大型游戏等都可以达到这个效果。

电脑无疑是属于这类东西。

大神Larry Wall 说程序员（黑客）有三大美德：

懒惰、傲慢、缺乏耐性。

这里的懒惰说的是为了让事情变得轻松一点而不辞辛苦

中文也有含义差不多的谚语——磨刀不误砍柴工。

使用电脑这样的智能设备，想实现某个结果都有一定的流程，很多时候这些流程里的某些步骤是重复的，

试想如果你能把某个常用的流程缩短了几步，那么每一次用这个流程你都省了一点儿时间，长久累积起来你能节省的时间就很可观了。

很长一段时间我都把时间花在这些可以提高效率的事情上，内心希望电脑能够用得更快一点、更高效、更省时间。

记得刚接触电脑时候的我非常迷它，想方设法去折腾系统，尝试各种软件，每天上网查一些教程、攻略，然后按着这个教程一步一步的实现，最终真的实现的话，那就非常的开心，如果不能够实现的话，就会钻进去出不来，可能是一个晚上、一天、甚至一周都在想这个问题是怎么回事。

很多时候，以至于我忘记了电脑只是一个工具，工具的本质是为了提高我们的工作学习效率。

之前买电脑是为了让我更好的学习、工作的等，是为了提高我的效率，没曾想会让自己沉迷于它的复杂结构中。

现在细想想折腾这些东西，给我带来了什么呢？

或许我是周围这批人最早就开始翻.墙上网的人，我也很早就开始折腾一些效率软件比如evernote、total command、listary、autohotkey等软件。

真能提高效率的话，还好，但很多时候这些软件的学习成本也非常的高，比如total command，可是至今我也用不习惯。

而且如果不得已的折腾，就非常的恼火。

比如昨天晚上电脑崩溃了，也不是崩溃了，是我崩溃了……

打开电脑的谷歌浏览器的时候网页加载不全，只能打开一些基本的html网页，图片也显示不出来，基本上是一篇空白。

我用测速软件测速，有网速啊，估摸着是网页被劫持了，下个火绒安全卫士，前几天看过keso推荐过这个软件，好像挺有情怀的，下载安装，检查，最终结果也没问题，可打开浏览器还是有问题，懵比了……

window电脑总是会出现一些莫名其妙难以理解的问题。

不得已下载了360，我想的是以毒攻毒，类似360这些软件，刚开始用过，但对它们印象不好，说是“清洁”电脑，可是自己却占据太多的电脑资源，也就是说，它把其它“电脑垃圾”清除了，然后好让它自己这个大佛占好多地方地方。

用流氓的手段对付流氓，应该可以，可以一通折腾，电脑重启了好多次，还是没用。

最终卸载浏览器，重新下载安装，终于正常。

但是，这里有个转折，其实换用另外一个浏览器也是可以用的，但是自己就钻进了牛角尖，没有解决这个问题，誓不罢休，一直到晚上快十二点才弄好，这个过程耗费了太大的心力了，好是憔悴。

基本上做不了什么事，本来想要发文章的，然后也没发，好内疚。

得到的教训就是尽可能的先用已有条件，把事情做完才是最重要的，电脑真的只是一个工具，是以为戒！
