使用markdown提高写作效率

发表于 2018-03-11 | | 本文总阅读量 次

很多东西，你会了就会感觉很简单，如果不会的话，那么就会像一堵墙一样搁在那边，感觉翻越过去好难。

但如果学会了，就会大大提高你的效率。

很多复杂软件就是这样的，比如“Autohotkey、workflow……‘

对于经常要和文字打交道的人，对于Word文档的编辑排版的繁琐肯定深有体会，比如行间距、字体大小、粗体……

我用markdown写文章也有一年多了，自己真的用得挺爽的，所以也是真心推荐给周围有写文章的朋友。

网络上有很多的相关的教程，作者写得也是深入浅出，可是看着那么多类似编程语言的符号，刚开始学习的同学，或许就望而却步，所以不才也写个简单的教程，希望帮到需要的人。

1、markdown简介

markdown是一种标记语言，我把它理解为“排版”语言，它实现的方式:

在文字前面加上一些特殊的符号，来告诉编辑器这个文字的格式是什么。

所以，在写字的时候，就不用关心排版了，在你写完的时候，排版也就完成了。

2、 基本语法

如果只是平常的写作，其实并不需要记住那么多的语法，只要把握住平常文章需要用到的几个常用的语法就可以了。

插入图片的话，现在的markdown编辑器，很多都可以直接实现平常word插入图片的方法，也是很方便的。

还有一些语法，比如表格、流程图、甘特图、代码框等，需要用到的时候再学习。

详细的语法可以查看这个链接： https://github.com/younghz/Markdown

3、编辑器的选择

我自己用的是Window系统，选择是typora，如果不习惯的话，可以选择其它的，可以参考少数派的这篇文章： https://sspai.com/post/42126

4、MarkdownHere的使用

这个李笑来老师写过一个教程，我觉得参考他写的，就可以很容易的使用了，网址在这里：http://t.cn/RE7vwRW

5、我的操作流程

我一般是在typora上码好字后，复制到微信公众号的编辑器中，然后使用插件“markdownhere”一键转换一下就好了，真正用于排版的时间，也就点击插件的时间而已。
